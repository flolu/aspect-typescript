# Usage

**Requirements**

- Linux
- Node.js
- pnpm
- Bazelisk

**Setup**

- `pnpm i` (Install Node dependencies)

**Commands**

- `bazelisk build //...` (Build everything)
- `bazelisk run //app:bin` (Run app)
